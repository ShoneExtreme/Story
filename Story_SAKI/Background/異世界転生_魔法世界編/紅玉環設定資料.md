
# 0. 概要
名称：紅玉環（こうぎょくかん）
位置：学園都市の一角に存在する、王国直営の合法娯楽街。
形状：三重の壁で区切られた縮小城郭型の娼館街。
特徴：娼婦は身分ではなく人気（＝売上・指名・落札）だけ上り詰めることができる完全実力主義。
内部は娼館や娼婦の住居、酒場、宿などで構成。外側は普通の町。

# 1. 三重壁構造と区画
- 中央核：最内側。宮殿然とした中枢建物群。紅玉后一人につき一つの宮殿が用意されている。
- 第一環：高級ゾーン。整然とした石畳・豪奢な建築。紅玉姫一人につき一つの宿が用意されている。
- 第二環：一般ゾーン。最も広く店舗数も最多。紅玉花一人につき一つの部屋を使用するが、どの立地の宿の部屋をとるかは早いもの順。
- 外縁（壁外）：普通の市街（宿屋・屋台・店等）。紅玉花区に面する側は特に賑わう。

## 入場と雰囲気
- 第二環（紅玉花区）：だれでも入場可。酒場・料理屋・宿屋・小劇場など一般客向けの施設が多い。
- 第一環（紅玉姫区）：だれでも入場可だがあらゆる店が高級。料金水準が跳ね上がる。
- 中央核（紅玉后区）：予約・許可制で原則関係者のみ。完全防衛・護衛同伴。

# 2. 階級（人気序列／週単位固定）
- 紅玉后（こう）：中央核所属の最上位。最大3人。
- 紅玉姫（き）：第一環所属の上位。最大100人。
- 紅玉花（か）：第二環所属の一般。登録だけを見れば1万人以上は存在する。
各自の翌週の階級は、日曜オークションの結果で決まる（固定期間：月〜土）。
成績が悪いからと言って即座に降格するわけではなく、悪い成績が1か月ほど下位に劣っている状態が続くと降格。
降格に合わせて、代わりとなる者が昇格する。

# 3. 従事者の身分と参加資格
- 刑奴（労従奴・犠従奴）：見目が良い者は強制従事。
- 従奴（常従奴・雅従奴）：契約者（主人）の判断で派遣可。
- 自由民（平民・貴族・王族）：本人の意思で志願可能。
※人気がすべてのため、平民でも王族でも人気に応じて花→姫→后へ上がり得る。

# 4. 営業日・稼働・日単価
- 営業日：月〜土のみ（6日）。日曜は全区画休業＋公開オークション。
- 稼働上限：1日1人（最大8時間）まで（一日単価制＝4hでも8hでも同額）。
- 8時間の扱い：館内プレイに加え、紅玉環内〜隣接市街での“デート”も公認（飲食・観劇・宿泊等）。
- 1週間の枠：各娼婦につき最大6日＝6ロット（月〜土）を日曜に競売。
- 最低落札額（階級別／日単価・入札スタート）
  - 紅玉花：1金貨〜
  - 紅玉姫：10金貨〜
  - 紅玉后：100金貨〜

# 5. 日曜オークション（週次・公開）
- 会場：紅玉環正門前広場。屋台・楽団が並び一日中開催。

  - 午前：人気薄の花を中心に数十人一斉。
  - 夕方：姫を少人数ずつ。
  - 夜：后を一人ずつ厳かな演出で。
- 観覧：だれでも見学可。ただし姫・后の至近観覧は有料（距離課金／額は未定）。花は至近無料。
- 落札単位：日ごと（6ロット）。重複予約は自動排除。
- 未落札者：翌週はフリー枠（当日先着契約・格安）。
- 料金目安：花＝10銀貨／姫＝1金貨／后＝10金貨（発生自体が大事件）

# 6. 予約・当日運用
- 週次確定：日曜の結果で月〜土の身分・配属・指名を確定。

# 9. 収益分配（最終）
- 自由民（平民・貴族・王族志願）：本人50%／国庫30%／施設20%

- 刑奴（労刑奴・犠刑奴）：本人0%／国庫100%（運営含む）
- 従奴は主人の投資（教育・衣装・医療）が重く、普段は屋敷・本業優先。紅玉環に出すのは“たまに・話題作り・閑散期のみが現実的。

# 10. 監査・治安・台帳
魔法刻印で本人確認・決済・予約ログを一元管理（改竄不可）。
ゲートは予約・許可の刻印照合で通過。

奴隷はへその魔法刻印で強制従順化（王国標準制度）。

# 11. 奴隷契約の制限
法人・団体による契約は不可。必ず個人名義でのみ契約可能。
奴隷を紅玉環へ出す場合も主人の名が公に出る（名誉・恥と直結）。

# 12. 紅玉祭（秋／花魁道中風・個別収益連動）

# 13. 特例解放称号：紅玉煌（こう）（永久称号）
趣旨：人気爆発により、個人ではレンタル契約が成立しないほどの超高額に達した従奴を、国が**平民へ昇格（奴隷籍抹消）**させる特例。
称号：紅玉煌（読みは后と同音のため、通俗的には**「あの“煌めく”方」**などと呼び分け）。
性質：一生ものの名誉称号。歴代で約10人のみ。
その後：多くは解放後およそ1年ほど紅玉環に自由民として出演して爆発的に稼ぎ、その後は引退・資産運用・事業等へ。称号は生涯存続。

# 14. 社会的ダイナミクス（要点）
花区：庶民も遊べる賑やか繁華街。娼婦目当てでない客も多数。
姫区：誰でも入れるが高級価格帯。利用そのものがステータス。
后区：国家級の見せ場。予約・護衛・秘匿性。
フリー枠：人気格差の可視化。案内所で誰でも確認・即契約可。
従奴派遣：年に数回レベル。成功すれば主人の名誉、失敗は恥。
人気＝運命：日曜の檻での一瞬が、翌週の地位と富を決める。
