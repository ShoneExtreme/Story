## 夜トレーニング

### 白桜式5種目ルーチン
| 曜日 | トレーニング種目 |
|------|-------------------------|
| 日曜 | プルアップ＋ベンチプレス＋レッグプレス（ジム） |
| 月曜 | リアレイズ |
| 火曜 | ルーマニアンデッドリフト |
| 水曜 | プッシュアップ |
| 木曜 | ブルガリアンスクワット |
| 金曜 | ワンハンドダンベルロウ |
| 土曜 | ショルダープレス |



### 白桜式5種目詳細

- プルアップ＋ベンチプレス＋レッグプレス（ジム）：ジムでトレーニングを行う。プルアップとベンチプレス、レッグプレスは必須種目とする。
- ルーマニアンデッドリフト：ハムストリング（大腿二頭筋、半腱様筋、半膜様筋）と脊柱起立筋をメインに鍛える。
- リアレイズ：三角筋後部をメインに、菱形筋も鍛える。
- ブルガリアンスクワット：直立姿勢で大腿四頭筋、大臀筋をメインに鍛える。
- プッシュアップ：大胸筋をメインに鍛える。
- ワンハンドダンベルロウ：広背筋・僧帽筋をメインに鍛える。- ワンハンドダンベルロウ：広背筋・僧帽筋をメインに鍛える。
- ショルダープレス：三角筋をメインに、大胸筋上部も鍛える。
- サイドレイズ：三角筋をメインに鍛える。
